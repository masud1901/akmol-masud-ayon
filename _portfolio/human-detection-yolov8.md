Developed a full-stack web application for detecting humans in user-uploaded images. Features a YOLO object detection model backend (FastAPI), a Streamlit frontend, and is fully containerized with Docker for easy deployment on an AWS EC2 instance.

**Technologies Used:** Computer Vision, FastAPI, Streamlit, Docker, AWS EC2
